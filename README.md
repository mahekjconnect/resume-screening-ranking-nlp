# AI-Powered Resume Ranking System 🚀

An intelligent screening tool that uses Natural Language Processing (NLP) to match candidate resumes with job descriptions based on semantic similarity.

## 📌 Project Overview
The goal of this project is to automate the initial stages of recruitment. By processing large datasets of resumes and job postings, the system ranks candidates based on how well their skills and experience align with specific job requirements.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, NLTK, Scikit-learn
* **Version Control:** Git & GitHub

## 📂 Data Foundation (Current Phase)
We have successfully implemented a robust text preprocessing pipeline to ensure high matching accuracy.

### Text Cleaning Features:
* **Noise Removal:** Stripping special characters, HTML tags, and redundant punctuation.
* **Normalization:** Case folding (lowercasing) for uniformity.
* **Stopword Filtering:** Removing common filler words (e.g., "the", "and") to focus on high-value keywords.
* **Lemmatization:** Reducing words to their dictionary root (e.g., "managing" -> "manage") using NLTK's WordNetLemmatizer.
